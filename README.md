# HDT Cardiovascular Cockpit

Dashboard untuk monitoring real-time 150+ parameter kardiovaskular menggunakan Human Digital Twin.

## Cara Menjalankan

1. **Install dependencies** (sudah selesai):
   ```bash
   npm install
   ```

2. **Jalankan development server**:
   ```bash
   npm run dev
   ```

3. **Build untuk production**:
   ```bash
   npm run build
   ```

4. **Preview production build**:
   ```bash
   npm run preview
   ```

## Teknologi yang Digunakan

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide React (icons)

## Fitur

- Real-time monitoring vital signs
- 3 Model: Geometrical, Physical, Data-Driven
- 150+ parameter kardiovaskular
- Interactive data mapping
- Responsive dashboard
